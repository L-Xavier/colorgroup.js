# resethtml
Reset the font size of HTML for REM.


# Usage

#### On your web page:

```
<script src="https://cdn.bootcss.com/jquery/3.2.1/core.js"></script>
<script src="js/htmltext.min.js"></script>
```

JS:

```
 $(document).ready(function(){
    resethtml();
});
```

#### Additional options:

```javascript
  zoom.to({
    x: 100,
    y: 200,
    width: 300,
    height: 300
  });
```

```javascript
  zoom.to({
    x: 100,
    y: 200,
    scale: 3
  });
```

#### Reset
```javascript
  zoom.out();
```
